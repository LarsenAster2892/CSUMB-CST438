Database Tables
Name Description
 statustypes : Entity 		Contains valid values for statustypes

A - Approved Timesheet
N - New Timesheet
R - Rejected Timesheet
S - Submitted Timesheet
 usertypes : Entity 		Contains valid values for usertypes:

A - Approver
I   - Instructor
S - Student
Z - Admin
 userprojects : Entity 		Links users to projects.  User can be on multiple projects and projects can have multiple users
 timesheetaction : Entity 		Service Learning TimeSheet Actions.  Contains list of actions that require response.
TimeSheet submit generates a record that requires approve or reject
TimeSheet reject generates a record that requires resubmit
 course : Entity 		Service Learning Courses
 timesheetdetail : Entity 		Service Learning Timesheet entry records  (1 for each start and end time on timesheet)
 timesheet : Entity 		Service Learning TimesSheets for students
 projectapprovers : Entity 		Service Learning TimeSheet Approvers for Projects.  Linked to Project via project id
 project : Entity 		Service Learning Projects.  Linked to course via course id
 useraccount : Entity 		Service Learning User accounts that can access the system
